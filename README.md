# Установка и запуск

## Требования

- Python 3.10 или новее
- Интернет для установки зависимостей

---

# Windows

## 1. Установить Python

Скачать Python:

https://www.python.org/downloads/

⚠️ При установке обязательно поставить галочку:

Add Python to PATH

---

## 2. Скачать проект

- Скачать ZIP с GitHub
- Распаковать в любую папку

---

## 3. Открыть терминал в папке проекта

Открыть папку проекта  
В адресной строке проводника написать:

cmd

и нажать Enter

---

## 4. Установить зависимости

pip install -r requirements.txt

Если не работает:

python -m pip install -r requirements.txt

---

## 5. Запустить программу

python main.py

---

# macOS / Linux

## 1. Проверить Python

Открыть Terminal и выполнить:

python3 --version

Если Python отсутствует — установить с python.org

---

## 2. Перейти в папку проекта

cd путь/к/папке/проекта

---

## 3. Установить зависимости

pip3 install -r requirements.txt

Если не работает:

python3 -m pip install -r requirements.txt

---

## 4. Запустить программу

python3 main.py
